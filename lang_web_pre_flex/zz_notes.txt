Conflicting Declarations - Ordering

Highest to Lowest

  Declarations
    Inline Style
      ID Selector (#)
        Class Selector (.) or Pseudo-Class (:)
          Element Selector (p, div, li, etc)
            Universal selector (*)

  You can override the above hierarchy with using
  !important;
